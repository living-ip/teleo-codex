# Resilience and Active Inference

## Core Argument
Resilience is polysemous — used to mean three distinct things in the literature. Active inference provides a formal framework that unifies all three:
1. **Inertia** — resistance to change → maps onto high precision beliefs (strong priors)
2. **Elasticity** — bouncing back to characteristic states → maps onto attracting states in free energy landscape
3. **Plasticity** — expanding the repertoire of good states → maps onto functional redundancy and structural degeneracy

## Key Claims for Extraction

### Three Senses of Resilience as Active Inference
- Inertia = high precision priors that resist updating (system maintains beliefs despite perturbation)
- Elasticity = homeostatic/allostatic return to characteristic states (low free energy attractors)
- Plasticity = structural degeneracy allowing multiple pathways to achieve same function (redundant generative models)

### Expected Free Energy Decomposes into Pragmatic and Epistemic Value
- Agents minimize expected free energy through two drives: pragmatic (preference satisfaction) and epistemic (information gain)
- Pragmatic value = achieving preferred outcomes
- Epistemic value = resolving uncertainty about the world
- This decomposition elegantly addresses exploration-exploitation tradeoff

### Precision as Confidence Invested in Beliefs
- Precision = inverse variance, quantifies confidence in a belief or model parameter
- High precision priors resist updating → inertia
- When precision is too high → rigid inference (fails to update given new evidence)
- When precision is too low → excessive sensitivity to noise (overfitting)
- Aberrant precision → psychopathology (delusions, depression)

### Generative Model Parameters (ABCDE Framework)
- A: likelihood matrix (observations → causes)
- B: transition matrix (state evolution over time)
- C: prior preferences (preferred observations)
- D: initial state priors
- E: policy priors (habitual preferences for action)
- Each parameter has associated precision

### Allostasis as Planning
- Homeostasis = reactive error correction
- Allostasis = anticipatory control (changing set-points proactively)
- Resilience in complex systems requires planning capacity (allostasis), not just reactive homeostasis

## Relevance to LivingIP
- **High priority.** The resilience framework maps directly onto knowledge base dynamics:
  - Inertia = stable high-confidence claims that resist change
  - Elasticity = belief cascading that restores coherence after new evidence
  - Plasticity = the collective's ability to hold multiple competing models (diversity as resilience)
- Precision weighting = confidence calibration (proven/likely/experimental/speculative)
- Aberrant precision = the failure mode where agents become too rigid (confirmation bias) or too flexible (no stable knowledge)
